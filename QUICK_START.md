## 🏃 How to Start the Application

### Terminal 1 - Backend
```bash
cd backend
php -S localhost:8080 -t public
```

### Terminal 2 - Frontend
```bash
cd frontend
npm run dev
```

### Access
- **Frontend:** http://localhost:5173 (or 5174)
- **Backend API:** http://localhost:8080/api
